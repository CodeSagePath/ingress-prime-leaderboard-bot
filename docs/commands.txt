Here's the list of all commands for your bot:

/submit - updates stored score
/leaderboard - returns top 10 ranks
/myrank - shows personal rank
/proof - optionally attaches screenshot saved locally with DB path
/start - Welcome message and registration prompt
/register - Register as an agent with codename and faction
/cancel - Cancel registration or verification process
/submit_data - Submit tab/space-separated data from Ingress Prime leaderboard
/top10 - Shows top 10 agents by cycle points
/top - Shows top 10 agents by cycle points for a specific faction (ENL or RES)
/lastcycle - Shows top 10 agents for the last cycle
/lastweek - Shows top 10 agents for the last 7 days
/privacy - Set group privacy mode (public, soft, strict)
/pending_verifications - Show all pending verification requests (admin only)
/approve_verification - Approve a verification request (admin only)
/reject_verification - Reject a verification request (admin only)
/backup - Manually trigger a backup (admin only)
/verify - Start verification process for submission
/help - Help message for the bot
/settings - Configure formatting settings for stats display
/broadcast - Broadcast a message to all users (admin only)
/stats - Show usage statistics (admin only)
